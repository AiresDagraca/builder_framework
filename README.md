**Dev Framework**
=================

A lightweight utility framework that provides a reusable set of CSS classes for developers to use across all projects, ensuring efficient workflows and a consistent, professional look.

**Overview**
------------

**Dev Framework** is a WordPress plugin designed to work seamlessly with **Oxygen Builder** and **Bricks Builder**.It enqueues a shared stylesheet containing layout utilities, components, and design tokens so developers can work faster and maintain consistency across sites.

This plugin is also integrated with **GitHub automatic updates** using the [plugin-update-checker](https://github.com/YahnisElsts/plugin-update-checker?utm_source=chatgpt.com) library.

**Features**
------------

*   Universal utility classes for spacing, layout, buttons, cards, grids, and more
    
*   Works in Oxygen Builder and Bricks Builder
    
*   Enqueued globally (front-end + builder context)
    
*   Automatic GitHub updates supported
    
*   Structured for easy expansion
    
*   Encourages consistent styling across all your client or personal projects
